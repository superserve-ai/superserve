Superserve gives you on-demand Firecracker micro-VMs that boot in under 200ms. Run code, transfer files, and snapshot VM state — then rollback or fork from any point in time.

Built for AI agent sandboxes, CI runners, and isolated dev environments.

## What you can do

<CardGroup cols={3}>
  <Card title="Run commands" icon="terminal">
    Execute shell commands with batch or streaming output
  </Card>
  <Card title="Transfer files" icon="file-arrow-up">
    Upload and download files to/from running VMs
  </Card>
  <Card title="Checkpoint" icon="camera">
    Snapshot full VM state (memory + disk) at any point
  </Card>
  <Card title="Rollback" icon="clock-rotate-left">
    Rewind a VM to any previous checkpoint
  </Card>
  <Card title="Fork" icon="code-fork">
    Clone running VMs into independent copies
  </Card>
  <Card title="SSH" icon="plug">
    Interactive terminal sessions via the CLI
  </Card>
</CardGroup>
