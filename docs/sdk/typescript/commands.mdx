---
title: Commands
description: Execute shell commands inside a sandbox, synchronously or with streaming output.
---

Commands are accessed via the `sandbox.commands` sub-module. Idle sandboxes are automatically resumed before execution.

## `sandbox.commands.run(command, options?)`

Execute a command and return the result. If `onStdout` or `onStderr` callbacks are provided, output is streamed via Server-Sent Events.

### Synchronous execution

```typescript
const result = await sandbox.commands.run("echo hello")
console.log(result.stdout)    // "hello\n"
console.log(result.stderr)    // ""
console.log(result.exitCode)  // 0
```

### Streaming execution

```typescript
const result = await sandbox.commands.run("npm start", {
  onStdout: (data) => process.stdout.write(data),
  onStderr: (data) => process.stderr.write(data),
  timeoutMs: 120_000,
})
```

### With environment and working directory

```typescript
const result = await sandbox.commands.run("python script.py", {
  cwd: "/app",
  env: { PORT: "3000", DEBUG: "1" },
  timeoutMs: 60_000,
})
```

### With cancellation

```typescript
const controller = new AbortController()
setTimeout(() => controller.abort(), 5000)

const result = await sandbox.commands.run("sleep 100", {
  signal: controller.signal,
})
```

## Options

| Field | Type | Default | Description |
|---|---|---|---|
| `cwd` | `string` | `/home/user` | Working directory. |
| `env` | `Record<string, string>` | — | Environment variables for this command. |
| `timeoutMs` | `number` | `30_000` | Command timeout in milliseconds. |
| `onStdout` | `(data: string) => void` | — | Stream stdout chunks. |
| `onStderr` | `(data: string) => void` | — | Stream stderr chunks. |
| `signal` | `AbortSignal` | — | Abort the command mid-execution. |

## Return type

```typescript
interface CommandResult {
  stdout: string
  stderr: string
  exitCode: number
}
```

## Handling non-zero exit codes

`run()` does not throw for non-zero exits — inspect `result.exitCode`:

```typescript
const result = await sandbox.commands.run("exit 42")
if (result.exitCode !== 0) {
  console.error(`Command failed: ${result.stderr}`)
}
```

It throws `TimeoutError` if the timeout elapses, or `NotFoundError` / `ConflictError` if the sandbox is unreachable.
