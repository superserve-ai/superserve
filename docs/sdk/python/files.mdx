---
title: Files
description: Upload and download files to and from a sandbox.
---

Files are accessed via `sandbox.files`. File operations go directly to the data plane (`boxd-{id}.sandbox.superserve.ai`) using the per-sandbox access token — the SDK handles this transparently.

## `sandbox.files.write(path, content, timeout=None)`

Write a file to the sandbox at the given absolute path. Parent directories are created automatically.

```python
# String content (encoded as UTF-8)
sandbox.files.write("/app/config.json", '{"key": "value"}')

# Binary content
with open("local-image.png", "rb") as f:
    sandbox.files.write("/app/image.png", f.read())
```

**Parameters:**

| Field | Type | Description |
|---|---|---|
| `path` | `str` | Absolute destination path (must start with `/`, no `..`). |
| `content` | `str \| bytes` | File contents. Strings are encoded as UTF-8. |
| `timeout` | `float` | Optional request timeout in seconds. |

## `sandbox.files.read(path, timeout=None)`

Read a file from the sandbox as raw bytes.

```python
data: bytes = sandbox.files.read("/app/image.png")
```

## `sandbox.files.read_text(path, timeout=None)`

Read a file from the sandbox as a UTF-8 string.

```python
text: str = sandbox.files.read_text("/app/config.json")
print(text)  # '{"key": "value"}'
```

## Round-trip example

```python
original = "hello, world!\n"

sandbox.files.write("/tmp/greeting.txt", original)
round_tripped = sandbox.files.read_text("/tmp/greeting.txt")

print(round_tripped == original)  # True
```

## Path rules

- Paths **must** start with `/`
- Paths **must not** contain `..` segments
- Parent directories are created automatically on write

## Errors

- `NotFoundError` — file does not exist (on read)
- `ValidationError` — invalid path
- `AuthenticationError` — access token invalid or sandbox deleted

## Async usage

```python
from superserve import AsyncSandbox

sandbox = await AsyncSandbox.create(name="async-example")
await sandbox.files.write("/tmp/data.txt", "hello")
content = await sandbox.files.read_text("/tmp/data.txt")
```
