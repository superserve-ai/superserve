<Info>
  **Prerequisites**:
  - Python 3.12 or higher
  - Docker (optional, for code sandbox features)
</Info>

## Get started in four steps

<Steps>
<Step title="Install RayAI">

```bash
pip install rayai
```

For sandbox support (code execution):
```bash
pip install rayai[sandbox]
```

</Step>

<Step title="Create a project">

```bash
rayai init my_project
cd my_project
```

This generates a project structure with:
- `agents/` - Directory for your agent code
- `pyproject.toml` - Project dependencies
- `.env.example` - Template for environment variables (copy to `.env`)

</Step>

<Step title="Create an agent">

```bash
rayai create-agent my_agent
```

Available frameworks: `python` (default), `langchain`, `pydantic` (more coming soon)

See the [framework guides](/frameworks/pydantic) for framework-specific implementations.

This creates `agents/my_agent/agent.py` with a template you can customize.

</Step>

<Step title="Run and test">

Start the server:
```bash
rayai up
```

Test your agent:
```bash
curl -X POST http://localhost:8000/my_agent/ \
  -H "Content-Type: application/json" \
  -d '{"query": "Hello!"}'
```

</Step>
</Steps>
