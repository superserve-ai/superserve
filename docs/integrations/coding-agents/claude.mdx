Claude Code is Anthropic's terminal-based coding agent. Run it inside a Superserve sandbox to give it a clean Linux environment for shell commands, package installs, and file edits, fully isolated from your local machine.

This integration guide is in progress. See [Quickstart](/quickstart) and [Run commands](/commands/overview) for the building blocks while we wire this up.
