Claude Code is Anthropic's terminal coding agent. Run it in a Superserve sandbox so its shell commands and file edits stay isolated from your machine.

## Via the console

1. Go to [console.superserve.ai](https://console.superserve.ai) and click **Create sandbox**.
2. Pick the **`superserve/claude-code`** template (Claude Code preinstalled).
3. Open **Advanced Options** and add `ANTHROPIC_API_KEY` under **Environment Variables**.
4. Create the sandbox, open its **Terminal**, and run `claude`.

## Via the SDK

```bash
npm install @superserve/sdk
export SUPERSERVE_API_KEY=ss_live_...
```

<CodeGroup>
```typescript TypeScript
import { Sandbox } from "@superserve/sdk"

const sandbox = await Sandbox.create({
  name: "claude-code",
  fromTemplate: "superserve/claude-code",
  envVars: { ANTHROPIC_API_KEY: "sk-ant-..." },
})
```

```python Python
from superserve import Sandbox

sandbox = Sandbox.create(
    name="claude-code",
    from_template="superserve/claude-code",
    env_vars={"ANTHROPIC_API_KEY": "sk-ant-..."},
)
```
</CodeGroup>

Open the sandbox in the [console](https://console.superserve.ai), launch the terminal, and run `claude`.

## Persist sessions

```typescript
await sandbox.pause()    // checkpoint state to disk
await sandbox.resume()   // restore exactly where you left off
```
