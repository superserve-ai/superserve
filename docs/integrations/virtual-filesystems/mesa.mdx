Mesa is a virtual filesystem layer. Mount it inside Superserve sandboxes for shared, persisted state across sandbox runs. Keep workspaces, caches, and project trees alive without baking them into a template.

This integration guide is in progress. See [Quickstart](/quickstart) and [Run commands](/commands/overview) for the building blocks while we wire this up.
