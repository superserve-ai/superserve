Hermes is NousResearch's open-source personal AI agent. Run it in a Superserve sandbox so its tools and shell commands stay isolated from your machine.

## Via the console

1. Go to [console.superserve.ai](https://console.superserve.ai) and click **Create sandbox**.
2. Pick the **`superserve/node-22`** template.
3. Open **Advanced Options** and add your model provider key (e.g. `ANTHROPIC_API_KEY`) under **Environment Variables**.
4. Create the sandbox, open its **Terminal**, and run:

   ```bash
   curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
   source ~/.bashrc
   hermes
   ```

## Via the SDK

```bash
npm install @superserve/sdk
export SUPERSERVE_API_KEY=ss_live_...
```

<CodeGroup>
```typescript TypeScript
import { Sandbox } from "@superserve/sdk"

const sandbox = await Sandbox.create({
  name: "hermes",
  fromTemplate: "superserve/node-22",
  envVars: { ANTHROPIC_API_KEY: "sk-ant-..." },
  timeoutMs: 100000,
})

await sandbox.commands.run(
  "curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash"
)
```

```python Python
from superserve import Sandbox

sandbox = Sandbox.create(
    name="hermes",
    from_template="superserve/node-22",
    env_vars={"ANTHROPIC_API_KEY": "sk-ant-..."},
    timeout_ms=100000,
)

sandbox.commands.run(
    "curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash"
)
```
</CodeGroup>

Open the sandbox in the [console](https://console.superserve.ai), launch the terminal, and run `hermes`.

## Keep it running

Start Hermes inside `tmux` so it survives closing the terminal or browser tab:

```bash
tmux new -d -s hermes 'hermes gateway'
```

Press `Ctrl+b` then `d` to detach. Reattach later with:

```bash
tmux attach -t hermes
```

## Persist sessions

```typescript
await sandbox.pause()
await sandbox.resume()
```
