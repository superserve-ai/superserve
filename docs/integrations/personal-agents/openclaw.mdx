OpenClaw is an open-source personal AI assistant. Run it in a Superserve sandbox so its actions and shell commands stay isolated from your machine.

## Via the console

1. Go to [console.superserve.ai](https://console.superserve.ai) and click **Create sandbox**.
2. Pick the **`superserve/node-22`** template.
3. Open **Advanced Options** and add your model provider key (e.g. `ANTHROPIC_API_KEY`) under **Environment Variables**.
4. Create the sandbox, open its **Terminal**, and run:

   ```bash
   npm i -g openclaw
   openclaw onboard
   ```

## Via the SDK

```bash
npm install @superserve/sdk
export SUPERSERVE_API_KEY=ss_live_...
```

<CodeGroup>
```typescript TypeScript
import { Sandbox } from "@superserve/sdk"

const sandbox = await Sandbox.create({
  name: "openclaw",
  fromTemplate: "superserve/node-22",
  envVars: { ANTHROPIC_API_KEY: "sk-ant-..." },
})

await sandbox.commands.run("npm i -g openclaw")
```

```python Python
from superserve import Sandbox

sandbox = Sandbox.create(
    name="openclaw",
    from_template="superserve/node-22",
    env_vars={"ANTHROPIC_API_KEY": "sk-ant-..."},
)

sandbox.commands.run("npm i -g openclaw")
```
</CodeGroup>

Open the sandbox in the [console](https://console.superserve.ai), launch the terminal, and run `openclaw onboard` to start the setup wizard.

## Persist sessions

```typescript
await sandbox.pause()
await sandbox.resume()
```
