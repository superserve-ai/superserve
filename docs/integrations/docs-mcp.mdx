Superserve publishes an MCP server at `https://docs.superserve.ai/mcp`. Connect it to any MCP-compatible AI assistant so it can search and read these docs without leaving your editor.

## Claude Desktop

Add to `~/Library/Application Support/Claude/claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "superserve-docs": {
      "url": "https://docs.superserve.ai/mcp"
    }
  }
}
```

Restart Claude Desktop and the assistant can answer questions about Superserve APIs, SDKs, and guides directly.

## Claude Code

```bash
claude mcp add superserve-docs https://docs.superserve.ai/mcp
```

## Codex

Add to `~/.codex/config.yaml`:

```yaml
mcpServers:
  superserve-docs:
    type: http
    url: https://docs.superserve.ai/mcp
```

## Cursor

Add to `.cursor/mcp.json` in your project root (or your global Cursor MCP config):

```json
{
  "mcpServers": {
    "superserve-docs": {
      "url": "https://docs.superserve.ai/mcp"
    }
  }
}
```

## Other MCP clients

Any client that supports remote HTTP MCP servers can connect using the server URL:

```
https://docs.superserve.ai/mcp
```
