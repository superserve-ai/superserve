The OpenAI Agents SDK is OpenAI's framework for building agents. Use Superserve sandboxes as your agent's tool runtime with fast cold-starts, per-session isolation, and full filesystem and shell access.

This integration guide is in progress. See [Quickstart](/quickstart) and [Run commands](/commands/overview) for the building blocks while we wire this up.
