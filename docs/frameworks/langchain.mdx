[LangChain](https://python.langchain.com/) provides chains, agents, and retrieval pipelines for building complex LLM applications. Any LangChain agent that reads input and prints output deploys to Superserve with no changes.

```python agent.py
from langchain_openai import ChatOpenAI
from langchain_core.messages import HumanMessage

llm = ChatOpenAI(model="gpt-4o")

while True:
    user_input = input()
    response = llm.invoke([HumanMessage(content=user_input)])
    print(response.content)
```

```txt requirements.txt
langchain-openai
```

```bash
superserve deploy agent.py
superserve secrets set my-agent OPENAI_API_KEY=sk-...
superserve run my-agent
```
