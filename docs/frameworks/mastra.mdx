[Mastra](https://mastra.ai/) is a TypeScript-first agent framework built on the [Vercel AI SDK](https://sdk.vercel.ai/). It works with any AI SDK-compatible model provider.

```typescript agent.ts
import { Agent } from "@mastra/core/agent"
import { openai } from "@ai-sdk/openai"
import { createInterface } from "readline"

const agent = new Agent({
  name: "assistant",
  instructions: "You are a helpful assistant.",
  model: openai("gpt-4o"),
})

const rl = createInterface({ input: process.stdin })
rl.on("line", async (line) => {
  const result = await agent.generate(line)
  console.log(result.text)
})
```

```json package.json
{
  "dependencies": {
    "@mastra/core": "latest",
    "@ai-sdk/openai": "latest"
  }
}
```

```bash
superserve deploy agent.ts
superserve secrets set my-agent OPENAI_API_KEY=sk-...
superserve run my-agent
```
