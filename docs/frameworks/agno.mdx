[Agno](https://docs.agno.com/) is a lightweight, model-agnostic Python framework for building agents. Simple API, minimal boilerplate.

```python agent.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(model=OpenAIChat(id="gpt-4o"))

while True:
    try:
        user_input = input()
    except EOFError:
        break
    response = agent.run(user_input)
    print(response.content)
```
