# superserve

Python SDK for the [Superserve](https://superserve.ai) sandbox API. Generated by [Fern](https://buildwithfern.com).

## Install

```bash
pip install superserve
```

## Usage

```python
from superserve import Superserve

client = Superserve(api_key="ss_live_...")

sandbox = client.sandboxes.create(snapshot="superserve/base")
result = sandbox.exec("echo 'Hello from Superserve!'")
print(result.stdout)
sandbox.stop()
```

See the [Superserve docs](https://superserve.ai) for full API reference.
