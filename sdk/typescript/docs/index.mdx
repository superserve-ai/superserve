Official TypeScript SDK for [Superserve](https://superserve.ai) hosted agents.

## Overview

The Superserve SDK provides a type-safe, modern TypeScript client for interacting with the Superserve API. It allows you to:

## Quick Start

<Steps>
  <Step title="Install the SDK">
    Install the package using your preferred package manager (see above).
  </Step>
  <Step title="Set up your API key">
    Get your API key from the [Superserve Dashboard](https://dashboard.superserve.ai) and set it as an environment variable:
    ```bash
    export SUPERSERVE_API_KEY=your-api-key
    ```
  </Step>
  <Step title="Create your first agent">
    Use the SDK to create and run an agent:
    ```typescript
    import { SuperserveClient } from '@superserve/sdk';

    const client = new SuperserveClient();

    const agent = await client.agents.create({
      name: 'my-assistant',
      model: 'claude-sonnet-4-20250514',
      systemPrompt: 'You are a helpful coding assistant.',
      tools: ['Bash', 'Read', 'Write', 'Glob', 'Grep']
    });

    console.log(`Created agent: ${agent.id}`);
    ```
  </Step>
  <Step title="Run the agent">
    Execute the agent with a prompt and get the output:
    ```typescript
    const output = await client.runs.run({
      agentId: agent.id,
      prompt: 'List all TypeScript files in the current directory'
    });

    console.log(output);
    ```
  </Step>
</Steps>

## Streaming Responses

For real-time output, use the streaming API:

<CodeGroup>
```typescript Node.js
const stream = await client.runs.stream({
  agentId: agent.id,
  prompt: 'Explain how async/await works'
});

for await (const event of stream) {
  switch (event.type) {
    case 'run.started':
      console.log('Run started:', event.runId);
      break;
    case 'message.delta':
      process.stdout.write(event.content);
      break;
    case 'run.completed':
      console.log('\nCompleted!');
      break;
  }
}
```

```typescript Browser
const stream = await client.runs.stream({
  agentId: agent.id,
  prompt: 'Explain how async/await works'
});

let output = '';
for await (const event of stream) {
  if (event.type === 'message.delta') {
    output += event.content;
    document.getElementById('output').textContent = output;
  }
}
```
</CodeGroup>

## Error Handling

The SDK provides typed error classes for different failure scenarios:

```typescript
import {
  NotFoundError,
  AuthenticationError,
  RunFailedError
} from '@superserve/sdk';

try {
  await client.runs.run({ agentId, prompt });
} catch (error) {
  if (error instanceof AuthenticationError) {
    console.error('Invalid API key');
  } else if (error instanceof NotFoundError) {
    console.error('Agent not found');
  } else if (error instanceof RunFailedError) {
    console.error(`Run failed: ${error.message}`);
  }
}
```

<Note>
  See the [Errors](/errors) documentation for a complete list of error types and handling patterns.
</Note>
