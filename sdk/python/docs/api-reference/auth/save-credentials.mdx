Saves API credentials to a JSON file.

## Signature

```python
def save_credentials_to_file(
    api_key: str,
    path: Path | None = None,
) -> None
```

## Parameters

<ParamField path="api_key" type="str" required>
  API key to save.
</ParamField>

<ParamField path="path" type="Path | None" default="None">
  Custom path for credentials file. Defaults to `~/.superserve/credentials.json`.
</ParamField>

## Behavior

- Creates the directory if it doesn't exist
- Sets restrictive file permissions (600 - owner read/write only)
- Overwrites existing file

## Example

```python
from pathlib import Path
from superserve_sdk import save_credentials_to_file

# Save to default location
save_credentials_to_file("sk_your_api_key")

# Save to custom location
save_credentials_to_file(
    "sk_your_api_key",
    path=Path("/custom/path/credentials.json"),
)
```

## File Format

```json
{
  "api_key": "sk_your_api_key"
}
```
