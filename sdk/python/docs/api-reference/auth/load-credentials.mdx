Loads API credentials from a JSON file.

## Signature

```python
def load_credentials_from_file(
    path: Path,
) -> Credentials | None
```

## Parameters

<ParamField path="path" type="Path" required>
  Path to credentials file.
</ParamField>

## Returns

<ResponseField name="Credentials | None" type="Credentials | None">
  Credentials object if file exists and is valid, otherwise None.
</ResponseField>

## Credentials Object

```python
class Credentials:
    api_key: str
```

## Supported Key Names

The function accepts these key names in the JSON file:
- `api_key`
- `token`
- `access_token`

## Example

```python
from pathlib import Path
from superserve_sdk import load_credentials_from_file

path = Path("~/.superserve/credentials.json").expanduser()
creds = load_credentials_from_file(path)

if creds:
    print(f"Found API key: {creds.api_key[:10]}...")
else:
    print("No credentials file found")
```

## clear_credentials

To remove stored credentials:

```python
from superserve_sdk import clear_credentials

# Clear default credentials
cleared = clear_credentials()
print(f"Cleared: {cleared}")

# Clear custom file
from pathlib import Path
cleared = clear_credentials(path=Path("/custom/creds.json"))
```
