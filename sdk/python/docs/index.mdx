The official Python SDK for [Superserve](https://superserve.ai) hosted agents.

## Overview

Superserve allows you to deploy and run AI agents in the cloud. This SDK provides a simple, type-safe interface for interacting with the Superserve platform.

## Installation

<Steps>
  <Step title="Install the SDK">
    Install using pip or uv:

    <CodeGroup>
    ```bash pip
    pip install superserve-sdk
    ```

    ```bash uv
    uv add superserve-sdk
    ```
    </CodeGroup>
  </Step>

  <Step title="Set up authentication">
    Set your API key as an environment variable:

    ```bash
    export SUPERSERVE_API_KEY=your-api-key
    ```

    Or save it to a credentials file:

    ```bash
    mkdir -p ~/.superserve
    echo '{"api_key": "your-api-key"}' > ~/.superserve/credentials.json
    ```
  </Step>

  <Step title="Create and run an agent">
    Start using the SDK in your Python code:

    <CodeGroup>
    ```python Async
    import asyncio
    from superserve_sdk import Superserve

    async def main():
        async with Superserve() as client:
            # Create an agent
            agent = await client.create_agent(
                name="my-assistant",
                model="claude-sonnet-4-20250514",
                system_prompt="You are a helpful assistant.",
            )
            print(f"Created agent: {agent.id}")

            # Run the agent
            run = await client.run(agent.id, "What is 2 + 2?")
            print(f"Output: {run.output}")

            # Clean up
            await client.delete_agent(agent.id)

    asyncio.run(main())
    ```

    ```python Sync
    from superserve_sdk import SuperserveSync

    with SuperserveSync() as client:
        agent = client.create_agent(name="sync-agent")
        run = client.run(agent.id, "Hello, world!")
        print(run.output)
        client.delete_agent(agent.id)
    ```
    </CodeGroup>
  </Step>
</Steps>

## Streaming Responses

Get real-time responses from your agents using streaming:

```python
import asyncio
from superserve_sdk import Superserve, MessageDeltaEvent

async def main():
    async with Superserve() as client:
        async for event in client.stream("my-agent", "Tell me a joke"):
            if isinstance(event, MessageDeltaEvent):
                print(event.content, end="", flush=True)
        print()

asyncio.run(main())
```

<Note>
  Streaming provides a better user experience for longer responses, as users can see output as it's generated rather than waiting for the complete response.
</Note>
